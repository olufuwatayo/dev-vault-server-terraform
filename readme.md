# Vault dev server

This script sets up a Vault dev server in Vagrant and creates a user with access to read and write secrets to it.
## Installation 

Ensure you have Vagrant installed
```bash
https://www.vagrantup.com/downloads.html
vagrant up
vagrant ssh
tail -f logfile.txt

```
